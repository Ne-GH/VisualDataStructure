# 使用指南
菜单选择不同数据结构或排序，暂时对以下内容做出部分支持：
* 数组
* 链表
* std::sort

左键单击选择某个节点，如果所选元素是数组则会选中所有节点（为了让数组始终排在一起不散开）  
左键双击某个元素可修改其值~~暂时并没有对输入内容进行判定，请输入int类型的数据~~。   
左键双击进入框选状态，可框选多个结点并移动   
右键显示操作菜单，可以选择添加 ~~和删除~~ ，右键菜单中的删除操作暂不可用，可直接通过右键需要删除的元素进行删除
